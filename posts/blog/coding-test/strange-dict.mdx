## 문제 출처
<a href="https://school.programmers.co.kr/learn/courses/30/lessons/120869" target="_blank" className="source">외계어 사전</a>

## 문제

1. **문제설명**
    - PROGRAMMERS-962 행성에 불시착한 우주비행사 머쓱이는 외계행성의 언어를 공부하려고 합니다. 알파벳이 담긴 배열 `spell`과 외계어 사전 `dic`이 매개변수로 주어집니다. `spell`에 담긴 알파벳을 한번씩만 모두 사용한 단어가 `dic`에 존재한다면 1, 존재하지 않는다면 2를 return하도록 solution 함수를 완성해주세요.
2. 제한사항
    - `spell`과 `dic`의 원소는 알파벳 소문자로만 이루어져있습니다.
    - 2 ≤ `spell`의 크기 ≤ 10
    - `spell`의 원소의 길이는 1입니다.
    - 1 ≤ `dic`의 크기 ≤ 10
    - 1 ≤ `dic`의 원소의 길이 ≤ 10
    - `spell`의 원소를 모두 사용해 단어를 만들어야 합니다.
    - `spell`의 원소를 모두 사용해 만들 수 있는 단어는 `dic`에 두 개 이상 존재하지 않습니다.
    - `dic`과 `spell` 모두 중복된 원소를 갖지 않습니다.
3. 입출력 예

| spell | dic | result |
| --- | --- | --- |
| ["p", "o", "s"] | ["sod", "eocd", "qixm", "adio", "soo"] | 2 |
| ["z", "d", "x"] | ["def", "dww", "dzx", "loveaw"] | 1 |
| ["s", "o", "m", "d"] | ["moos", "dzx", "smm", "sunmmo", "som"] | 2 |

## 수도코드 작성

1. dic에서 각각의 element를 순회한다.
2. 순회한 element에서 각 철자가 spell을 모두 가지고 있는지 확인한다. 
    
    2-1. dic의 각 철자를 순회하여 spell이 가지고 있는 value의 index를 가진 배열로 반환한다.
    
    2-2. 이후 반환 된 배열에서 -1이 없고, 겹치는 수가 없는지 확인한다. 두 조건을 만족하면 spell 배열의 길이와의 일치 여부를 학인하고 true를 반환한다.
    
3. 이후 true가 하나라도 있으면, 1, 없으면 2를 반환한다.

## 코드 작성

```jsx
function solution(spell, dic) {
  const answer = dic.map((item) => {
    const check = item.split("").map((char) => {
      return spell.indexOf(char);
    });
    return (
      check.filter(
        (item, index, arr) => item !== -1 && index === arr.indexOf(item)
      ).length === spell.length
    );
  });
  return answer.includes(true) ? 1 : 2;
}
```

## 보안점

- 한번에 이해하기 쉽지 않은 코드다.
- spell의 요소를 각각 따로 볼 필요가 없이 하나의 문자열로 일치 여부를 파악했다면 더욱 쉽게 코드를 작성할 수 있었다.
```js
function solution(spell, dic) {
  const answer = dic.map((item) => {
    return [...item].sort().toString() === spell.sort().toString();
  });
  return answer.includes(true) ? 1 : 2;
}
```
