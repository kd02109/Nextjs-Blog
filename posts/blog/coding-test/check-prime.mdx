## 문제 출처
<a href="https://school.programmers.co.kr/learn/courses/30/lessons/120878" target="_blank"className="source">유한소수 판별하기</a>


## 문제 설명
- 소수점 아래 숫자가 계속되지 않고 유한개인 소수를 유한소수라고 합니다. 분수를 소수로 고칠 때 유한소수로 나타낼 수 있는 분수인지 판별하려고 합니다. 유한소수가 되기 위한 분수의 조건은 다음과 같습니다.
    - 기약분수로 나타내었을 때, 분모의 소인수가 2와 5만 존재해야 합니다.
    
    두 정수 `a`와 `b`가 매개변수로 주어질 때, a/b가 유한소수이면 1을, 무한소수라면 2를 return하도록 solution 함수를 완성해주세요.
    

### 제한사항

- `a`, `b`는 정수
- 0 < `a` ≤1,000
- 0 < `b` ≤ 1,000

---

## 의사코드 작성

- a의 모든 약수 구하기
- a의 가장 큰 약수부터 b와 나누어 떨어지는지 확인하고 나누어졌다면, break
- a의 가장 큰 약수로 나눈 b가 2와 5로만 나누어 떨어지면 1을 반환 그렇지 않으면 2를 반환.

---

## 코드

```jsx
function solution(a, b) {
    let number = a/b
    if(number === parseInt(number)) return 1
    
    // a의 약수 구하기 
    const arrA = []
    for(let i=2; i<=a; i++){
        if(a%i===0){
            arrA.push(i)
        }
    }

    //b 구하기
    let numB =b
    const {length} = arrA
    for(let i=length-1; i>=0; i--){
        if(numB % arrA[i] === 0) numB/=arrA[i] 
        if(numB<=a) break
    }

    //b가 2,5로만 이루어진 수인지 구한기
    let answer = 1
    while (numB !== 1){
        if(numB %2 ===0){
            numB/=2
        }else if(numB %5 ===0){
            numB/=5
        }else{
            answer = 2
            break
        }
    }
    return answer
}
```

## 보완

- a의 모든 약수를 찾을 필요 없이 a와 b의 최대 공약수만을 찾으면 된다.
- 다른 사람의 코드를 보니 더 쉽게 이해가 되었다.

```js
function solution(a, b) {
    let n = 1;
    for (let i = 1; i <= Math.min(a,b); i++) {
        if (a%i===0 && b%i===0) n = i;
    }

    b/=n;
    while (b%2===0) b/=2;
    while (b%5===0) b/=5;

    return b === 1 ? 1 : 2;   
}
```
