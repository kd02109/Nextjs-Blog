## 문제 출처

<a href="https://school.programmers.co.kr/learn/courses/30/lessons/120849" target="_blank" className="source">모음제거</a>

### **문제 설명**

영어에선 a, e, i, o, u 다섯 가지 알파벳을 모음으로 분류합니다. 문자열 `my_string`이 매개변수로 주어질 때 모음을 제거한 문자열을 return하도록 solution 함수를 완성해주세요.

### 제한 사항

- `my_string`은 소문자와 공백으로 이루어져 있습니다.
- 1 ≤ `my_string`의 길이 ≤ 1,000

### 입출력 예 설명

입출력 예 1

- "bus"에서 모음 u를 제거한 "bs"를 return합니다.

입출력 예 2

- "nice to meet you"에서 모음 i, o, e, u를 모두 제거한 "nc t mt y"를 return합니다.


### 나의 코드

```jsx
function solution(my_string) {
  const stringArr = my_string.split("");
  const AEIOU = ["a", "e", "i", "o", "u"];
  const notAEIOU = stringArr.filter((item) => {
    for (let i = 0; i < AEIOU.length; i++) {
      if (AEIOU[i] === item) {
        return false;
      }
    }
    return true;
  });

  return notAEIOU.join("");
}
```

- 문자열을 배열로 반환하여 filter를 처리해서 문제를 풀었다. 이때 filter내에서 a,e,i,o,u 여부를 모든 문자에 대해서 반복문으로 판단했다.

### 개선 1

```jsx
function solution(my_string) {
	const stringArr = my_string.split("");
	const AEIOU = ["a", "e", "i", "o", "u"];
	const notAEIOU = stringArr.filter((item) => {
		return !AEIOU.includes(item);
		});
	return notAEIOU.join("");
}
```

- `array.prototype.includes()` 를 통해 매우 쉽게 true, false인지를 파악할 수 있었다. 굳이 반복문을 filter 콜백 함수 내에서 실행할 필요가 없었다.

### 개선 2

```jsx
function solution(my_string) {
	return my_string.replace(/[aeiou]/g, "");
}
```

- RegExp 정규 표현식을 사용하면 한 줄로 해결을 해버린다!! 어제 정규 표현식을 공부했지만, 바로 떠올리지 못하는 것을 보면, 아직 이 녀석과 친해지기에는 멀었구나라는 생각을….. 😢
