## 문제 출처
<a href="https://school.programmers.co.kr/learn/courses/30/lessons/120876" target="_blank"className="source">겹치는 선분의 길이</a>

- 선분 3개가 평행하게 놓여 있습니다. 세 선분의 시작과 끝 좌표가 [[start, end], [start, end], [start, end]] 형태로 들어있는 2차원 배열 `lines`가 매개변수로 주어질 때, 두 개 이상의 선분이 겹치는 부분의 길이를 return 하도록 solution 함수를 완성해보세요. `lines`가 [[0, 2], [-3, -1], [-2, 1]]일 때 그림으로 나타내면 다음과 같습니다.
- 선분이 두 개 이상 겹친 곳은 [-2, -1], [0, 1]로 길이 2만큼 겹쳐있습니다.

### 제한사항

- `lines`의 길이 = 3
- `lines`의 원소의 길이 = 2
- 모든 선분은 길이가 1 이상입니다.
- `lines`의 원소는 [a, b] 형태이며, a, b는 각각 선분의 양 끝점 입니다.
    - 100 ≤ a < b ≤ 100

# 의사코드 작성

1. 항상 시작 부분의 값이 작다. 더불어 각 선분의 인덱스를 모두 표시하면 총 200개이다. 
2. 0인 값이 200개 있는 빈 배열을 생성한다.
3. 각 선분의 시작점과 끝점을 구하고 시작점부터 끝점-1까지 선이 지나갔다는 표시를 하기 위해 생성한 빈 배열에 +=1을 한다.
4. 모든 선분에서 3의 과정을 반복한다.
5. 배열에서 값이 2이상인 부분인 곳만 추려내어 배열의 길이를 반환한다.


```js
function solution(lines) {
    const arr = Array.from({length:200}).fill(0)
    for(let i =0; i<lines.length; i++){
        let left = lines[i][0]
        let right = lines[i][1]
        for(let j=left; j<right; j++){
            arr[j+100] +=1
        }
    }
    const answer = arr.filter(item=> item>1).length
    return answer;
}
```
